# jengo-knowledge-public

Shareable knowledge patterns and published learnings.

## What's Here

| Path | Purpose |
|------|---------|
| `patterns/*.md` | Published patterns with documented evidence and behavioral tests |

## Architecture

This repo is part of the **Jengo 3-Tier Modular System**:
- **Layer:** knowledge (what Jengo knows — memories, patterns, learnings)
- **Tier:** public (safe to share, publish, or include in open-source)

This is the knowledge commons — patterns that have been validated through multiple instances and are useful to the broader community. Source patterns originate in `jengo-knowledge-private`.
